Introducción a Git

Git es un sistema de control de versiones que permite gestionar el código fuente de proyectos de software de manera eficiente.

Comandos Básicos de Git

  • git init: Inicializa un repositorio Git en el directorio actual.
  • git add <archivo>: Añade un archivo al área de preparación.
  • git commit -m "mensaje": Guarda los cambios realizados en el repositorio.
  • git push: Sube los cambios locales al repositorio remoto.
  • git pull: Obtiene cambios del repositorio remoto y los fusiona con el repositorio local.
